# 💈 Sistema de Gerenciamento - Barbearia

Este projeto foi desenvolvido em **Python** com o objetivo de simular o funcionamento de uma barbearia, permitindo o gerenciamento de clientes, agendamentos, serviços e pagamentos através de um sistema interativo no terminal.

O sistema segue o conceito de **CRUD (Create, Read, Update, Delete)**, possibilitando organizar e controlar informações de forma simples e eficiente.

---

## 🚀 Funcionalidades

O sistema é dividido em módulos principais:

### 👤 Cadastro de Clientes
- Registro de nome, telefone e e-mail
- Confirmação de cadastro com mensagem de sucesso

### 📅 Agendamento de Serviços
- Escolha de serviços disponíveis
- Seleção de preferências do cliente
- Confirmação de agendamento

### 📋 Consulta de Agendamentos
- Visualização de horários disponíveis
- Seleção de horário desejado
- Confirmação mediante senha

### 💇 Serviços e Preços
- Lista completa de serviços oferecidos:
  - Corte
  - Barba
  - Corte + Barba
  - Luzes
  - Platinado
  - Alisamento
  - Corte Infantil
  - Barboterapia
  - Limpeza de Pele
  - Pezinho
  - Pigmentação
  - Hidratação Capilar

### ✂️ Escolha de Barbeiro
- Seleção entre profissionais disponíveis
- Confirmação da escolha

### 💳 Pagamentos
- Opções disponíveis:
  - Débito
  - Crédito
  - Pix
- Confirmação de pagamento

### ⭐ Avaliações
- Cliente pode avaliar o serviço prestado
- Sistema simples de estrelas (1 a 5)

### 🔔 Notificações
- Consulta de notificações do cliente

---

## 🛠️ Tecnologias e Estruturas Utilizadas

O projeto foi desenvolvido utilizando conceitos fundamentais de Python:

- **Entrada de dados (`input`)**
  - Interação com o usuário via terminal

- **Laços de repetição (`while`)**
  - Navegação contínua no menu

- **Condicionais (`if/elif/else`)**
  - Controle de fluxo do sistema

- **Variáveis**
  - Armazenamento de dados temporários
